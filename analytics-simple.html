📊 FUSSBALLBOT ANALYTICS SETUP

🚀 SCHNELLSTART

1. Google Analytics 4 erstellen

GA4 öffnen

→ Konto erstellen → Property: "fussballbot.qzz.io"

2. Google Tag Manager erstellen

GTM öffnen

→ Container: "fussballbot.qzz.io" → Web

3. IDs kopieren

GA4: G-XXXXXXXXXX
GTM: GTM-XXXXXXX

4. In HTML eintragen

Öffne fussballbot-brutal.html und ersetze:

// Suche nach: G-XXXXXXXXXX → Ersetze mit deiner GA4 ID
GTM-XXXXXXX → Ersetze mit deiner GTM ID

5. Live schalten

git add fussballbot-brutal.html
git commit -m "Add Analytics"
git push origin main

🔧 TECHNISCHE IMPLEMENTATION

Automatisch getrackte Events:

1. telegram_click - Telegram Button Klicks
2. premium_click - Premium Button Klicks (99 CHF)
3. scroll_25/50/75/90 - Scroll-Tiefe
4. page_view - Alle Seitenaufrufe

Conversion Goals in GA4:

• Telegram Engagement
• Premium Conversion
• Content Engagement (90% Scroll)

📈 DASHBOARD EINRICHTUNG

GA4 Berichte konfigurieren:

1. Echtzeit - Aktive Nutzer

2. Engagement - Seitenaufrufe, Verweildauer

3. Monetarisierung - Premium Conversions

4. Nutzer - Demografie, Geräte, Standorte

Conversion Funnel:

Landing Page → Scroll 50% → Telegram Click → Premium Click
